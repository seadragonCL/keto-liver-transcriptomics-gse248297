# Genes & Nutrition Submission Checklist

Based on the Genes & Nutrition research-paper instructions.

## Required Manuscript Sections

- Title page with study-design signal in title
- Full author names and institutional addresses
- Corresponding author
- Structured abstract, max 350 words:
  - Background
  - Results
  - Conclusions
- 3-10 keywords
- Background
- Methods
- Results
- Discussion
- Conclusions
- List of abbreviations
- Declarations:
  - Ethics approval and consent to participate
  - Consent for publication
  - Availability of data and materials
  - Competing interests
  - Funding
  - Authors' contributions
  - Acknowledgements
  - Authors' information, optional

## Formatting Reminders

- Use double line spacing in the submitted manuscript file.
- Add line and page numbering before submission.
- Do not use page breaks in the manuscript.
- Use SI units.
- Number tables and figures in order of first citation.
- Keep figure titles under 15 words and legends under 300 words.
- Put figure titles and legends in the manuscript, not only in figure files.
- Include URLs as numbered references, not only inline links.
- Public datasets supporting the conclusions should be referenced in the reference list.

## Items Still Needed Before Submission

- Confirm final author list, affiliations, author order, and corresponding-author email.
- Deposit analysis code and derived tables in a public repository, ideally GitHub plus Zenodo DOI.
- Replace placeholder repository URL in the Availability of data and materials section.
- Confirm funding and competing-interest statements.
- Decide whether to retain the generative-AI disclosure; if yes, keep it transparent and concise.
- Convert the draft to DOCX or LaTeX for submission.
- Upload figure files individually or as required by the submission portal.
- Consider adding an independent validation dataset before submission if aiming for a stronger review outcome.
